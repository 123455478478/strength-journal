# 个人力量训练记录 App 产品设计

## 1. 产品定位

这是一个个人使用的健身房力量训练记录器。它不制定训练计划、不安排每周训练频率，也不判断用户今天应该练什么。

核心任务：

> 用户开始一次自由训练，选择实际进行的动作，准确记录每组重量、次数、时间和组间间歇，并能随时回看历史。

## 2. 首版范围

- 点击“开始健身”建立训练并记录开始时间，点击“结束健身”记录结束时间与总时长。
- 从动作库选择动作，也可以创建自定义动作。
- 记录动作名称、主要/辅助肌群、实际组数、每组重量和次数。
- 每组通过“开始本组”和“结束本组”自动记录时间；结束后再填写实际重量和次数。
- 结束健身后的总结页可保存文字心得、状态或下次调整事项。
- 点击历史日历日期，可新增、修改或删除当天的训练记录。
- 自动计算每组持续时间、动作内组间间歇和动作总用时。
- 查看训练历史、单次训练详情和单个动作趋势。
- 根据最近训练记录、常用动作和当前已选肌群实时排列建议动作。
- 数据完全保存在本地，支持导出和恢复。

## 3. 明确不做

- 周训练计划、训练日程和训练模板。
- 自动推荐今天练什么。
- 自动增加重量或生成训练方案。
- 课程、饮食、社区、排行榜和会员系统。
- 摄像头姿态识别和自动计数。

## 4. 导航结构

底部使用四个入口：

1. **训练**：开始自由训练，或恢复未完成训练。
2. **动作**：按胸部、背部、肩部、手臂、腿和核心筛选动作，并查看动作要点。
3. **历史**：通过训练日历和时间列表查看记录。
4. **我的**：个人档案、力量/次数极限、单位、备份与恢复。

## 5. 自由训练流程

1. 点击“开始健身”，从此刻开始记录本次训练总时长。
2. 从动作库添加本次实际进行的动作。
3. 进入动作，点击“开始第 1 组”。
4. 完成该组后点击“结束本组”，再填写实际重量、次数并保存；可使用重量 ±1、次数 ±1 的快捷按钮。
5. App 开始显示实时组间间歇。
6. 点击“开始下一组”时保存上一段组间间歇。
7. 最后一组完成后结束该动作，继续添加或选择其他动作。
8. 点击“结束健身”，保存训练结束时间和总时长；首页训练卡片也提供快捷结束入口。
9. 在结束总结页填写本次训练的文字记录并完成。

训练不要求预先确定动作数量、顺序或组数。用户可以随时加动作、加组、删组或跳过。
新组默认沿用上一组的重量和次数；动作第一组参考最近一次训练中同动作第一组的记录。未开始的额外组支持左滑删除。

## 6. 必备记录字段

| 字段 | 记录方式 |
| --- | --- |
| 动作名称 | 从动作库带入或用户自定义 |
| 针对肌群 | 动作资料带入，可修改主要/辅助肌群 |
| 组数 | 根据实际完成组自动统计 |
| 次数 | 每组填写或沿用上一组 |
| 重量 | 每组填写或沿用上一组，支持 kg/lb |
| 时间 | 每组开始与完成时自动记录 |
| 间歇 | 下一组开始时间减去上一组结束时间 |

## 7. 计时定义

```text
本组持续时间 = 本组结束时间 - 本组开始时间
第 n 次组间间歇 = 第 n+1 组开始时间 - 第 n 组结束时间
动作总用时 = 最后一组结束时间 - 第一组开始时间
训练总用时 = 训练结束时间 - 训练开始时间
```

第一组之前和最后一组之后的时间不属于动作的组间间歇。动作总用时包含动作内部的全部组间间歇。

## 8. 训练记录结构

```text
训练 WorkoutSession
└── 动作记录 ExerciseLog
    └── 组记录 SetLog
```

### WorkoutSession

- id
- 开始时间、结束时间、总用时
- 动作数量、完成组数
- 训练备注

### ExerciseLog

- 动作 ID
- 动作名称及肌群快照
- 在本次训练中的顺序
- 第一组开始时间、最后一组结束时间、动作总用时
- 实际完成组数

### SetLog

- 组序号
- 重量、次数
- 开始时间、结束时间、持续时间
- 与上一组之间的实际间歇
- 热身组/正式组标记和备注

历史记录保存当时的名称和肌群快照，之后修改动作资料不能改变旧记录。

## 9. 异常恢复

- 每次开始或完成一组都立即保存。
- App 切到后台、锁屏或关闭后，使用时间戳恢复当前状态。
- 误触开始可以取消本组。
- 忘记点击完成时允许修正开始和结束时间，并重新计算相关间歇。
- 删除已完成组需要确认，并重新计算动作时间。

## 10. 建议动作

建议动作只改变首页动作排列，不替用户制定计划。规则分成两个阶段：

- 今天尚未开始第一组：读取前两次训练，硬性避开上一次覆盖的部位，并优先推荐近两次覆盖较少的部位；同一部位不会连续成为训练前建议。
- 第一组已经开始：找到最近开始动作的所属肌群，只推荐同一肌群的相关动作。
- 同一候选范围内，再根据最近八次记录中的动作使用频率排列常用动作。
- 当前训练已经添加的动作不会重复推荐。

每条建议显示简短原因，例如“近两次未练背部”“避开上次胸部”或“继续本次背部训练”。建议计算在本机实时完成。

## 11. 动作资料

动作库的一级分类为胸部、背部、肩部、手臂、腿和核心；二级分类包括：

- 胸部：上胸、胸部整体、胸部孤立。
- 背部：垂直拉、水平拉。
- 肩部：前束/整体、中束、后束、肩袖。
- 手臂：肱二头、肱三头、前臂。
- 腿：股四头、腘绳肌、臀部、小腿。
- 核心：腹部、抗伸展、侧腹、抗旋转、腰部和腰部稳定。

每个动作包含器械、目标肌群、负重倾向、常用次数范围和三条关键注意事项。负重和次数只作为一般训练信息，不是针对个人的处方。

动作库现包含 67 个常见动作，后续可以继续从开放动作数据集导入名称、器械和肌群资料。

## 12. 历史与个人档案

- 当月日历标出存在训练记录的日期。
- 点击任意日期进入当日记录，可新增当日训练或修改、删除已有记录。
- 记录卡片支持从右向左滑动显示删除按钮，删除前必须确认。
- 日历下按时间倒序排列训练记录。
- 每条记录展示日期、主要肌群、动作名称、完成组数和总用时。
- 个人档案记录身高、体重、年龄、性别。
- 极限记录包括卧推 1RM、深蹲 1RM、引体向上连续次数和俯卧撑连续次数。
- 所有字段输入后自动保存到本机。

## 13. 首版验收标准

用户能够：

1. 开始一场没有预设内容的自由训练。
2. 搜索并添加一个动作。
3. 记录多组重量、次数、持续时间和组间间歇。
4. 临时增加或删除动作与组。
5. 中断并恢复正在进行的训练。
6. 完成训练并查看全部记录。
7. 按日期和动作查看历史表现。
8. 导出并恢复个人数据。
9. 根据历史偏好看到可解释的建议动作。
10. 按五大肌群和二级部位浏览动作要点。
11. 在日历上识别训练日期，并按顺序查看记录。
12. 保存并再次打开个人身体数据和极限成绩。
